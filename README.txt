First compile the programs using following command on your linux terminal


             gcc algorithms.c getDataFromFile.c main.c memoryListFunctions.c myheader.h


Then run the file using following ways


	./a.out -i <input_file_name> -t <type_of_algorithm>



<input_file_name> :-


In this file contains data about processes and their require memory

e.g.: files/process_3.txt

#this file contains process information anf their memory needed
#comments can be given by # sign

#Process    Memory-required(bytes)

   5		3985
   6		488696
   7		8439893
   8		575456
   9		787564
   10		2425454
   11		255474534
   12		246634
   13		585345
   14		834656

#please do not edit this file





<type_of_algorithm> :-

	1) ALLMEMORY
	2) FCFS
	3) PAGING
	4) SEGMENTATION



*****************************************************************
examples:-


./a.out -i "files/process_2.txt" -t "ALLMEMORY"


./a.out -i "files/process_1.txt" -t "FCFS"


./a.out -i "files/process_3.txt" -t "PAGING"


./a.out -i "files/process_3.txt" -t "SEGMENTATION"



you can redirect or write the output to a file using > character

e.g:


./a.out -i "files/process_3.txt" -t "SEGMENTATION" > output.txt
